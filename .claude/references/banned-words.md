# Mots et formulations bannis

Ces expressions sont des tics de langage IA. Ne les utilise **jamais** dans un article SFEIR.

## Mots interdits

| Mot interdit | Alternatives |
|---|---|
| "crucial" | "essentiel", "clé", "central" |
| "landscape" / "paysage" (au sens figuré) | reformule |
| "leverage" / "exploiter" (au sens positif) | "utiliser", "tirer parti de" |
| "robust" / "robuste" | "solide", "fiable" |
| "seamless" / "transparent" (au sens de sans friction) | "fluide", "sans accroc" |
| "delve" / "plonger dans" | "explorer", "regarder de plus près" |
| "tapestry" / "tapisserie" | reformule |
| "game-changer" | reformule concrètement |
| "paradigm" / "paradigme" | "approche", "modèle" |

## Formulations interdites

| Formulation interdite | Correction |
|---|---|
| "Il est important de noter que..." | Va droit au fait |
| "Il convient de souligner que..." | Dis-le directement |
| "Dans le paysage actuel de..." | Attaque le sujet |
| "Force est de constater que..." | Constate directement |
| "N'hésitez pas à..." | Utilise l'impératif directement |
| "Il va sans dire que..." | Si ça va sans dire, ne le dis pas |
| "En fin de compte..." | "Au final", ou rien |
| "Cela étant dit..." | Enchaîne directement |
| "Sans plus attendre..." | Commence directement |
