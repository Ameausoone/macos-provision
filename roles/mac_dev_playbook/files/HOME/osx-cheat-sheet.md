# macOS

## Finder shortcuts

```
Cmd + Shift + G     go to path in any Finder window
Cmd + Shift + .     toggle hidden files
```

## oh-my-zsh: tab management

```sh
tab               # open current directory in new tab
split_tab         # split tab horizontally
vsplit_tab        # split tab vertically
```

## oh-my-zsh: Finder integration

```sh
ofd               # open current directory in Finder
pfd               # print path of frontmost Finder window
pfs               # print current Finder selection
cdf               # cd to current Finder directory
pushdf            # pushd to current Finder directory
```

## Files

```sh
quick-look <file>   # Quick Look a file
man-preview <cmd>   # open man page in Preview.app
showfiles           # show hidden files
hidefiles           # hide hidden files
mktemp -d           # create temp directory
```

## Terminal recording

```sh
asciinema rec demo.cast    # record terminal session
asciinema play demo.cast   # replay
agg demo.cast demo.gif     # convert to GIF
```

## Misc

```sh
spotify             # control Spotify from CLI
open -a "App Name"  # open app from terminal
pbcopy / pbpaste    # clipboard from CLI
```
