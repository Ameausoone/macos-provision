# Generate the CA bundle and test npm (Zscaler)

## Generate the bundle (Zscaler CA + Node default roots)

```bash
mkdir -p ~/.certs
node -p 'require("tls").rootCertificates.join("\n")' > ~/.certs/node-roots.pem
cat ~/.certs/zscaler.pem ~/.certs/node-roots.pem > ~/.certs/npm-ca-bundle.pem
```

## Configure npm to use the bundle

```bash
npm config set cafile "$HOME/.certs/npm-ca-bundle.pem"
npm config set strict-ssl true
```

## Test npm

```bash
npm ping
npm view markdownlint-cli2 version
npm config get cafile
cat ~/.certs/npm-ca-bundle.pem | head -n 3
```
